Customised version of Openfire Meetings for Webinars

Overview

Webinar plugin for Openfire is a customised version of Openfire Meetings for Webinars. By Default, Openfire Meetings uses Jitsi Meet and Jitsi Videobridge in a NxN conference mode where everyone speaks and listens to each other. For Webinars, we need only a few talkers and everyone else to be listeners.

Known Issues

Installation

  1. Unzip the webinar.jar file to a temporary WEBINAR_PLUGIN folder.
  2. Copy all the webinar*.jar files in the WEBINAR_PLUGIN/classes folder to the OPENFIRE_HOME/lib folder.
  3. Copy the webinar.jar file to the OPENFIRE_HOME/plugins folder.
  4. Restart Openfire.
  5. Configure the admin properties page.

Configuration

Under WebApps settings -> Webinar tab you can configure.

How to use

  1. Presenter should use URL https://your_openfire_meetings_server:7443/webinar/your_room?mode=presenter
  2. Attendee should use URL https://your_openfire_meetings_server:7443/webinar/your_room?mode=attendee
